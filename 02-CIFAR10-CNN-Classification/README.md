# CIFAR-10 Image Classification using CNN

## Objective

This project builds a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset into one of ten object categories using TensorFlow and Keras.

---

## Dataset

The project uses the **CIFAR-10** dataset, which consists of:

- 50,000 training images
- 10,000 testing images
- RGB images of size **32 × 32 pixels**
- 10 different classes

### Classes

- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

---

## Libraries Used

- TensorFlow
- Keras
- NumPy
- Matplotlib

---

## Methodology

1. Loaded the CIFAR-10 dataset.
2. Normalized image pixel values.
3. Built a Convolutional Neural Network (CNN).
4. Trained the model using the training dataset.
5. Evaluated model performance on the test dataset.
6. Visualized training and validation accuracy.
7. Predicted image classes using the trained model.

---

## Results

- Model: Convolutional Neural Network (CNN)
- Dataset: CIFAR-10
- Output: Multi-class image classification

The trained model successfully classifies images into one of the ten CIFAR-10 categories with good accuracy.
