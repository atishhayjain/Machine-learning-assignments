# Machine Learning and AI Assignments

This repository contains all nine past Machine Learning and Artificial Intelligence assignments assigned between **20 May 2026 and 19 July 2026**.

The assignments cover machine learning, deep learning, computer vision, reinforcement learning, recommendation systems, deployment, and Retrieval-Augmented Generation (RAG).

Assignments will be added to this repository one by one as they are completed:

1. Adult Census Income Classification  
2. CIFAR-10 Image Classification using CNN  
3. Face Recognition using CNN (LFW Dataset)  
4. Cancer Detection using MRI Images  
5. CartPole Reinforcement Learning Agent Training  
6. LunarLander Reinforcement Learning Agent Training  
7. Movie Recommendation System  
8. End-to-End Render Deployment Project  
9. RAG Chatbot — Capstone Project  

Each assignment folder will contain the source code and/or Google Colab notebook, along with its own README explaining the objective, dataset, methodology, and results.
